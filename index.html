Esta URL solo confirma que npm run dev está corriendo. No es tu sitio WordPress.

  1. En wp-config.php: define('VITE_DEV_SERVER', true);
  2. Deja este comando en marcha en la carpeta del tema: npm run dev
  3. Abre el sitio en el navegador (p. ej. http://jlb-school.local), no esta pestaña

Prueba rápida: /src/main.js debe mostrar código JavaScript (respuesta 200).
